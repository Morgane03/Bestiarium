## 1. Créer un compte utilisateur
Pour créer un compte utilisateur, suivez ces étapes :
1. Ouvrez l'application Bestinarium.
2. Cliquez sur **S'inscrire**.
3. Entrez un nom d'utilisateur unique (par exemple, "Jean123"), une adresse e-mail valide, et un mot de passe sécurisé.
4. Cliquez sur **Créer un compte**.

Si l'inscription est réussie, vous recevrez une confirmation de l'enregistrement de votre compte.

**Exemple de message** :  
`Utilisateur créé avec succès.`

---

## 2. Se connecter à votre compte
Une fois inscrit, vous pouvez vous connecter avec votre nom d'utilisateur et votre mot de passe :
1. Ouvrez l'application Bestinarium.
2. Cliquez sur **Se connecter**.
3. Entrez votre nom d'utilisateur et mot de passe.
4. Cliquez sur **Se connecter**.

Si la connexion est réussie, vous serez redirigé vers votre tableau de bord où vous pourrez gérer vos créatures et participer à des batailles.

**Exemple de message** :  
`Utilisateur connecté avec succès.`

---

## 3. Gérer vos créatures

### Ajouter une créature
Pour ajouter une créature à votre collection, suivez ces étapes :
1. Cliquez sur **Ajouter une créature**.
2. Entrez le type de créature (par exemple, "Dragon") et le nombre de têtes (par exemple, 1).
3. Cliquez sur **Ajouter**.

Si l'ajout est réussi, votre créature apparaîtra dans votre liste.

**Exemple de message** :  
`Créature ajoutée avec succès.`

### Afficher une créature
Vous pouvez consulter les informations détaillées d'une créature en cliquant sur **Afficher les détails** à côté du nom de la créature dans votre liste.

### Créer une créature hybride
Vous pouvez créer une créature hybride en combinant deux créatures existantes. Pour ce faire :
1. Cliquez sur **Créer une créature hybride**.
2. Sélectionnez deux créatures existantes.
3. Cliquez sur **Fusionner**.

Une fois la fusion terminée, la nouvelle créature hybride apparaîtra dans votre collection.

---

## 4. Participer à des batailles
Pour organiser un combat entre deux de vos créatures, suivez ces étapes :
1. Cliquez sur **Ajouter un match**.
2. Sélectionnez deux créatures à affronter.
3. Cliquez sur **Lancer le combat**.

Le combat sera simulé automatiquement et vous recevrez les résultats, y compris la créature gagnante et les statistiques du match.

**Exemple de message** :  
`Match ajouté avec succès. Le gagnant est le Dragon des Cieux.`

---

## 5. Déconnexion
Lorsque vous avez terminé votre session, vous pouvez vous déconnecter en cliquant sur **Déconnexion** dans le menu de votre profil.

**Exemple de message** :  
`Utilisateur déconnecté avec succès.`

---

## Notes supplémentaires
- **Sécurisez votre compte** : Utilisez des mots de passe uniques et difficiles à deviner.
- **Hybrides spéciaux** : Certaines créatures hybrides peuvent avoir des pouvoirs spéciaux que vous ne trouverez pas chez les créatures classiques.
